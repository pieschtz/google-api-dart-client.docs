Gets the information about the current user along with Drive API settings

  • includeSubscribed Whether to include subscribed items when calculating the number of remaining change IDs Default: true.

  • maxChangeIdCount Maximum number of remaining change IDs to count Default: 1.

  • startChangeId Change ID to start counting from when calculating number of remaining change IDs
