Code List<AccessConfig> accessConfigs #

Array of configurations for this interface. This specifies how this interface is configured to interact with other network services, such as connecting to the internet. Currently, ONETOONE_NAT is the only access config supported. If there are no accessConfigs specified, then this instance will have no external internet access.
