Code String iPProtocol #

Required; this is the IP protocol that is allowed for this rule. This can either be a well known protocol string (tcp, udp or icmp) or the IP protocol number.

String iPProtocol;

Code List ports #

An optional list of ports which are allowed. It is an error to specify this for any protocol that isn't UDP or TCP. Each entry must be either an integer or a range. If not specified, connections through any port are allowed. Example inputs include: "22", "80,"443" and "12345-12349".
