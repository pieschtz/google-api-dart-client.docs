Code String id #

The ID of the entry. For entries of type "person" or "circle", this is the ID of the resource. For other types, this property is not set.

core.String id;

Code String type #

The type of entry describing to whom access is granted. Possible values are: - "person" - Access to an individual. - "circle" - Access to members of a circle. - "myCircles" - Access to members of all the person's circles. - "extendedCircles" - Access to members of everyone in a person's circles, plus all of the people in their circles. - "public" - Access to anyone on the web.
