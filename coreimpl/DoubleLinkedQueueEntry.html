DoubleLinkedQueueEntry<E> Class

An entry in a doubly linked list. It contains a pointer to the next entry, the previous entry, and the boxed element.

E remove() {
  _previous._next = _next;
  _next._previous = _previous;
  _next = null;
  _previous = null;
  return _element;
