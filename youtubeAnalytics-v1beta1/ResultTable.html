Code List<List<Object>> rows #

Contains all rows of the result table. Each row contains an array with the values for the columns. The order matches the order of the column information provided in the "columnHeaders" field. If no data is available for the given query, the "rows" element will be omitted from the response. The response for a query with the day dimension will not contain rows for the most recent days.

core.List<core.List<core.Object>> rows;
